# Install SwapRAM
 ```html
wget -q -O /usr/bin/swapram "https://raw.githubusercontent.com/NevermoreSSH/swapram/main/swapram.sh" && chmod +x /usr/bin/swapram && swapram
  ```
Then Just type command below
 ```html
swapram
  ```

## Description : For VPS only!!!

 SwapRAM Options:
- 512MB
- 1GB
- 2GB
- 4GB
